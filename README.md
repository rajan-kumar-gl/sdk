### How to install
 replace `{{GEM_NAME}}`, `{{version}}`,  and `{{path}}` value and  add following command to you `Gemfile`. 
 
```ruby 
gem {{GEM_NAME}}, '{{version}}'', git: 'https://github.com/greatlearning/sdk', glob: '{{path}}'
```
